**********************
*
* 1     Images
*
**********************

Ci-joints, dans le dossier "Mockups (écrans de l'application)", les images correspondant à tous les Mockups (écrans) de l'application Proxibanque_V3.
Il y a en tout 16 écrans différents dans l'application, dont 5 écrans d'erreurs dans un sous-répertoire "Ecrans Erreur" (erreur d'authentification, erreur de montant négatif, etc...).


**********************
*
* 2      Visionnage interactif
*
***********************

1 - Installation du logiciel "balsamiq"

	Si balsamiq n'est pas installé, vous pouvez le télécharger à l'adresse suivante:
	https://balsamiq.com/download/

2 - Double-cliquer sur le fichier "Proxibanque_v3_maquette.bmpr" pour l'ouvrir dans le logiciel "balsamiq".

	Sur le volet de gauche, cliquer sur le mockup "Index" correspondant à la page d'accueil de l'application.
	Taper sur la touche F5, pour rendre les écrans interactifs.
	Vous pouvez alors "naviguer" (sans pouvoir cependant rentrer de données), en cliquant sur les différents boutons de l'application.
	Note: les 5 écrans d'erreurs n'apparaissent pas en mode de lecture interactive.
	Pour sortir de cette navigation interactive, taper sur la touche ECHAP.
